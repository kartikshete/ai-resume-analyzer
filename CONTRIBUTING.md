# Contributing to AI Resume Analyzer

We welcome community contributions to this project!

## Development Setup
- **Client**: `cd client && npm install && npm run dev`
- **Server**: `cd server && python -m venv venv && source venv/bin/activate && pip install -r requirements.txt && python main.py`

## How to Contribute
1. Fork the repo and create your branch from `main`.
2. Add features or bug fixes.
3. Ensure the code is readable and followed local styles.
4. Submit a Pull Request with a clear description of the change.

## Project Vision
To provide a simple, local tool that helps candidates understand how ATS systems parse their resumes.
